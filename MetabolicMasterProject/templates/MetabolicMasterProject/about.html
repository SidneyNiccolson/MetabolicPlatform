Metabolic's Online Platform


The Metabolic's Online Platform is designed to let users:

  • Learn about nutrient recovery from waste through the BioCascadeModeller web-application. This tool is still under development and the main idea behind it is to let users design there nutrient recovery trajectory and model the output with different treatment technologies.
  • Analyse there energy usage from offices or there houses. This tool is not implemented as of yet and depends on Metabolic's sensory system project called SmartShit. If let say a office is implemented with SmartShit chips, users will be able to visually see there energy usage.

    • This Web-application is built with Django 1.5.4 and Python 2.7.8
